## Running Locally

Clone the project

```bash
git clone https://github.com/MarioMH8/terminal.mariomh.com.git
```

Go to the project directory

```bash
cd terminal.mariomh.com
```

Install dependencies

```bash
yarn install
```

Start the server

```bash
yarn dev
```
